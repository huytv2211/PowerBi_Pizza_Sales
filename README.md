# PowerBi_Pizza_Sales

# PROBLEM STATEMENT

## KPI's REQUIREMENT

Để phân tích hiệu quả kinh doanh từ dữ liệu bán pizza, chúng ta có thể tập trung vào một số chỉ số KPI quan trọng. Đây là những thước đo giúp đánh giá hiệu suất và tìm ra cơ hội cải thiện:

- **Total Revenue**  
  Tổng doanh thu = tổng giá trị của tất cả đơn hàng pizza.

- **Average Order Value (AOV)**  
  Giá trị trung bình mỗi đơn hàng = tổng doanh thu / tổng số đơn hàng.

- **Total Pizzas Sold**  
  Tổng số lượng pizza đã bán = tổng cộng số lượng pizza trong tất cả đơn hàng.

- **Total Orders**  
  Tổng số đơn hàng được đặt.

- **Average Pizzas Per Order**  
  Số pizza trung bình mỗi đơn hàng = tổng số pizza bán ra / tổng số đơn hàng.

## PROBLEM STATEMENT CHARTS REQUIREMENT

- **Daily Trend for Total Orders**  
  → Bar chart hiển thị số lượng đơn hàng theo từng ngày, giúp nhận diện biến động theo ngày.

- **Hourly Trend for Total Orders**  
  → Line chart thể hiện số đơn hàng theo giờ trong ngày, giúp xác định giờ cao điểm.

- **Sales by Pizza Category**  
  → Pie chart phân bổ doanh thu theo từng loại pizza, cho thấy loại nào phổ biến nhất.

- **Sales by Pizza Size**  
  → Pie chart thể hiện tỷ lệ doanh thu theo kích thước pizza, phản ánh sở thích khách hàng.

- **Total Pizzas Sold by Category**  
  → Funnel chart so sánh số lượng pizza bán ra theo từng loại.

- **Top 5 Best Sellers**  
  → Bar chart hiển thị 5 loại pizza bán chạy nhất theo Revenue, Quantity, Orders.

- **Bottom 5 Worst Sellers**  
  → Bar chart hiển thị 5 loại pizza bán kém nhất theo Revenue, Quantity, Orders.
